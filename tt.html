pseudoelementos ::

::first-letter

solo modifica el primer elemento o la primera letra

::first-line

se utiliza para modificar la primera linea del div, no importa el tamaño, la declaracion se va adaptando

::selection

se usa cuando se selecciona un elemento, por ejemplo, en los navegadores la letra se ponen blancas y el background azul

::placeholder

habla por si solo, modifica algunas cosas del placeholder

::marker

se utiliza para modificar los marcadores

pseudo clases :

:hover

se utiiza cuando el puntero esta encima del elemento

:active

cunado damos clickk

li:first-child

se utiliza en una lista para selecionnar el primer elemento

li:last-child

se utiliza para el ulitmo

li:nth-child(n)

se utiliza para seleccionar algun otro elemento

:empty

funciona para los elementos vacios y desaparecen

input[type="checkbox"]:checked

podemos modificar los input para cambiarle el estilo dependiendo del contenido dentro

:valid/invalid

se utiliza para cuando inputs tipo mail no tiene una sintaxis valida
